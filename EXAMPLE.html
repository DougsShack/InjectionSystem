Welcome to DSIS Example

This page demonstrates Doug's Shack Injection System (DSIS) in action.

How It Works

The header and footer on this page are automatically injected from separate HTML files:

  • Header comes from: dsis/examplehead.html
  • Footer comes from: dsis/examplefoot.html

You can also use callbacks to execute code when the component is loaded. This is done by adding the data-dsis-callback attribute to the <div> element. In this example the callback function is callbackTest.

Try It Yourself

To use DSIS in your own pages:

  1. Include <script src="dsis/dsis.js"></script> in your head section
  2. Add <div id="dsis-componentname"></div> where you want components to appear
  3. Create your component files in the dsis/ folder

Error Example

This is an example of an error that will be displayed if the component file is not found. The error is also logged to the browser console.

You can use the CSS class dsis-error to style the error message.
